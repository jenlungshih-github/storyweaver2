# **App Name**: Children Story Teller

## Core Features:

- Story Template: Interactive form with fields for characters, setting, plot points, and moral lessons to help create a detailed story outline.
- AI-Powered Prompt Generator: Generates unique children's story ideas based on specified themes and keywords. It will operate as a tool which will suggest a limited set of parameters and options the user must explicitly confirm. Includes diverse cultural elements from global folk tales. 你是一個講兒童故事的專家，熟悉世界的民間童話故事，並且擅長畫兒童繪本。
- AI Story Expander: Takes a user-created story outline and expands it into a full children's story with engaging narrative and descriptions using generative AI. A Save Story button to save the story.
- Feedback Tool: Provides actionable feedback on user drafts. Offers concrete examples.
- Story Saver: Persists user-created stories and outlines using Firebase Firestore. Every story will be saved in Traditional Chinese and English version for two records.
- Illustration Generation: If needed, use an AI image generator to make relevant illustrations. A link to open a new window and link to Gemini.
- Multi-Language Support: Includes support for both English and Traditional Chinese. Default to Traditional Chinese.

## Style Guidelines:

- Primary color: Warm, inviting yellow (#FFD700) evoking the warmth of childhood memories.
- Background color: Very light, desaturated yellow (#FAFAD2) creates a soft, comforting atmosphere.
- Accent color: Gentle orange (#FFA500) complements the yellow tones.
- Body and headline font: 'Literata' for an elegant, literary, vintage feel.
- Use whimsical, hand-drawn style icons related to fairy tales and storytelling. For example, wands, books, animals, or castles.
- Use a clean, simple layout with ample white space. Structure stories with a traditional Intro-Problem-Adventure-Resolution.
- Incorporate gentle, playful animations on button hovers and page transitions.
